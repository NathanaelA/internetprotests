Apache Instructions

This requires your Apache webserver has Mod-Rewrite enabled; you need to add the following block of code: 
    # Required set of rewrite rules
    Include apache_ips.conf
Please make sure you copy the ips list to the /etc/apache2/banned-ips file. If your apache2 directory is located elsewhere use the proper path on your server. You typically put this in your main config file for apache so it will effect all sites on your server.
